== Description ==

Displays your friend's recent activity on your dashboard page and helps you communicate and stay in touch with them.

If you want to add some friends, just add it through Link tab and the plugin will works for those links.


== Installation ==


1. Upload the zipped file to yoursite/wp-content/plugins
1. Activate the plugin through the 'Plugins' menu in WordPress
